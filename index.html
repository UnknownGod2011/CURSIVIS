Text Pitch:

Cursivis addresses a practical problem in modern workflows: even with powerful AI tools, users still lose time and focus switching applications, typing prompts, and manually applying results. This friction is especially visible in professional environments where users already rely heavily on the mouse for precision and speed.

Cursivis is a context-aware plugin built for the Logitech MX ecosystem that allows users to act directly on what they select on screen using a deliberate hardware trigger. When a user selects text, code, an image, or a visual element and presses an MX control, Cursivis analyzes both the selected content and the application context, then performs the most relevant action automatically.

For example, selecting code in an editor triggers bug analysis and prepares a clean fix; selecting a long report generates a concise summary; selecting foreign-language text produces an instant translation; selecting an email draft improves clarity and tone; and hovering over a design color reveals exact HEX or RGB values. The same trigger works across applications, but the result changes based on context.

If the user wants a different outcome, holding the MX trigger reveals a small set of context-aware options through an Actions Ring, allowing refinement without leaving the current workflow. This interaction is fast, consistent, and requires no prompt writing or mode switching.

By combining precise pointing with intentional hardware input and the Actions SDK, Cursivis delivers immediate, visible value while remaining realistic to implement and scale. It fits naturally into Logitech's professional ecosystem and demonstrates how hardware and software can work together to reduce friction and improve productivity in everyday workflows.
